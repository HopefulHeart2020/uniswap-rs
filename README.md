# uniswap-rs

Simple library for interacting with Uniswap v2 and v3 smart contracts.

## Getting started

Add this to your cargo.toml:

```toml
[dependencies]
uniswap = { git = "https://github.com/danipopes/uniswap-rs" }
```

Check out the examples for implementation details.
